# TravelTrucks - service where you can find any car for yourself

This is a web application for searching cars.

## About the project

This application helps users search and campare vans using convenient filters such as:

- location
- transmission type
- air conditioning
- additional features (etc.)

You can also:

- view details vachive information
- read reviews
- book a vechive online
- add a vechive to favorites

## Tech stack

- **React**
- **Redux**
- **AXIOS** for API requests
- **FORMIK** (forms)
- **Mui (@mui/material)** (UI components)
- **Bootstrap icons**

## Getting started

1. Install dependencies:

   ```bash
   npm install

   ```

2. Run the project locally:
   npm run dev
